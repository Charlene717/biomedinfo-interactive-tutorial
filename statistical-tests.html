STEP 6 / 9

統計假設檢定

挑對檢定法是分析的關鍵分岔點——選錯會讓所有後續推論失效。

Choosing the right test is the analytical fork in the road — pick wrong and every downstream inference is voided.

概覽

假設檢定的本質

所有經典統計檢定都遵循同一邏輯：(1) 建立虛無假設 H₀ 與對立假設 H₁；(2) 假設 H₀ 為真，計算觀察數據的檢定統計量；(3) 算出在 H₀ 下觀察到「至少這麼極端」結果的機率（p 值）；(4) 若 p < α 則拒絕 H₀。

三大判斷要素決定該用哪種檢定：

  • 變數類型：連續、二元、類別、順序
  • 組數：單組、兩組、多組
  • 樣本關係：獨立 vs. 配對

同時要檢驗該檢定的前提條件（常態性、變異數同質性、獨立性）。違反 → 改用無母數版本或穩健方法。

All classical tests follow the same logic: (1) state a null H₀ and alternative H₁; (2) compute a test statistic under H₀; (3) compute the probability of seeing data at least this extreme (p-value); (4) reject H₀ if p < α.

Three factors decide the test:

  • Variable type: continuous, binary, categorical, ordinal
  • Number of groups: one, two, or many
  • Sample relationship: independent vs. paired

Also check assumptions (normality, equal variance, independence). Violations → use non-parametric or robust alternatives.

效應量

三、p 值之外：效應量與信賴區間

2016 ASA p 值聲明強調：p 值不能單獨衡量證據強度。報告檢定結果時，應同時呈現：

  • 點估計（差異、OR、相關係數）
  • 95% 信賴區間——區間越窄越精確
  • 效應量——Cohen's d、Cliff's δ、η²、Cramer's V
  • p 值 作為輔助

2020 後越來越多期刊（如 Nature 系列）要求作者完全省略「statistically significant」用詞，改用效應量+CI 描述。

The 2016 ASA p-value statement stresses: p alone is not a measure of evidence strength. Always report:

  • Point estimate (difference, OR, correlation)
  • 95% confidence interval — narrower = more precise
  • Effect size — Cohen's d, Cliff's δ, η², Cramer's V
  • p-value as a supplement

Post-2020, many journals (including Nature family) ask authors to drop "statistically significant" and instead use effect size + CI.

程式碼

實作範例

# Two-sample t-test + effect size
library(effsize)
data(sleep)
fit <- t.test(extra ~ group, data=sleep, var.equal=FALSE)
cohen <- cohen.d(extra ~ group, data=sleep)
cat("t=",fit$statistic,"p=",fit$p.value,
    " 95% CI: [",fit$conf.int,"]\nCohen's d =",cohen$estimate,"\n")

# Mann-Whitney U (non-parametric alt)
wilcox.test(extra ~ group, data=sleep, conf.int=TRUE)

# Chi-square + Fisher
tab <- matrix(c(30,10,15,25), nrow=2)
chisq.test(tab)        # standard
fisher.test(tab)       # when expected counts < 5
import numpy as np
from scipy import stats

# Welch's t-test (default in scipy when equal_var=False)
a = np.array([5.2,6.1,5.8,7.0])
b = np.array([4.2,4.9,5.0,4.6])
t, p = stats.ttest_ind(a, b, equal_var=False)

# Mann-Whitney U
u, p = stats.mannwhitneyu(a, b, alternative="two-sided")

# Chi-square + Fisher
chi2,p,dof,exp = stats.chi2_contingency([[30,10],[15,25]])
odds,p_f = stats.fisher_exact([[30,10],[15,25]])
import pingouin as pg

# t-test with effect size and Bayes factor
pg.ttest(a, b, correction="auto")
# columns: T, dof, alternative, p-val, CI95%, cohen-d, BF10, power

# Normality + Levene's variance test in one line
pg.normality(a); pg.homoscedasticity([a,b])

# One-way ANOVA + post-hoc
pg.anova(dv="score", between="group", data=df)
pg.pairwise_tukey(dv="score", between="group", data=df)

📝 隨堂測驗

Q1. 三組獨立、連續、非常態，用？

A. One-way ANOVA
B. Mann-Whitney U
C. Kruskal-Wallis
D. Friedman
Kruskal-Wallis 是 ANOVA 的無母數版本，適合 ≥3 獨立組的非常態資料。Kruskal-Wallis is the non-parametric analog of ANOVA for ≥3 independent groups.

Q2. 2x2 期望次數 = 3 應該用？

A. Pearson chi-square
B. Fisher's exact
C. McNemar
D. 獨立 t-testIndependent t-test
期望次數 < 5 時 chi-square 的近似不夠好，改用 Fisher exact 計算精確機率。When expected counts < 5, chi-square's approximation is poor — use Fisher's exact for precise probabilities.

Q3. p = 0.04 正確的解讀？

A. 虛無假設是真的機率為 4%There's a 4% chance H₀ is true
B. 若 H₀ 為真，觀察到此資料或更極端結果的機率為 4%Assuming H₀ true, the probability of seeing data this extreme (or more) is 4%
C. 結論犯型一錯誤的機率為 4%There's a 4% chance of making a Type I error
D. 效應量為 0.04The effect size is 0.04
p 值是條件機率：P(資料 | H₀ 為真)。它不是 H₀ 為真的機率。p is a conditional probability: P(data | H₀ true). It is NOT the probability that H₀ is true.

Q4. 前後配對連續且常態？

A. 兩獨立樣本 t-testIndependent t-test
B. Paired t-test
C. Mann-Whitney U
D. Chi-square
配對 = 同一受試者前後測，應用 paired t-test（檢驗差異是否為 0）。Paired = same subject pre/post → paired t-test (testing whether differences differ from 0).
