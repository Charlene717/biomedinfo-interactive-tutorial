樣本量與統計檢定力

α、β、效應量與樣本量四者互為條件。理解這四者的關係，能避免「白做研究」或「過度浪費資源」。

α, β, effect size, and sample size are mutually constraining. Master their relationship to avoid wasted studies — or wasted resources.

概覽

為什麼樣本量計算這麼重要？

樣本量太少 → 檢定力不足，真有的效應抓不到（type II error 升高）。
樣本量太多 → 浪費受試者時間、實驗經費、倫理負擔，甚至偵測到「統計顯著但臨床無意義」的微小差異。
因此 ICH-E9、CONSORT、STROBE、ARRIVE 等指引都要求論文與計畫書「事前」報告樣本量計算依據。

Too few subjects → underpowered studies miss real effects (high type II error).
Too many → waste of subjects, money, and ethical burden, plus detection of statistically significant but clinically meaningless differences.
ICH-E9, CONSORT, STROBE, and ARRIVE all require a pre-specified sample size justification.

四要素

一、四個互相牽動的參數

α — 顯著水準

當虛無假設 H₀ 為真時錯誤拒絕的機率。常設 0.05。雙尾比單尾保守。基因組研究常用 5×10⁻⁸（Bonferroni-corrected for ~1M tests）。

The probability of falsely rejecting H₀ when true. Typically 0.05. Two-sided is more conservative than one-sided. Genome-wide studies use 5×10⁻⁸ (Bonferroni for ~1M tests).

β — 型二錯誤；1-β = 檢定力

當 H₁ 為真卻沒拒絕 H₀ 的機率。標準是 80% 檢定力 (β=0.2)，臨床試驗常用 90%。

Probability of failing to reject H₀ when H₁ is true. Standard target is 80% power (β=0.2); clinical trials often use 90%.

Effect size — 效應量

差異的大小。Cohen's d (連續變數)、OR/RR (二元變數)、HR (存活)、ρ (相關)。越小越難偵測，需越大樣本。

Magnitude of the difference. Cohen's d (continuous), OR/RR (binary), HR (survival), ρ (correlation). Smaller effects → larger samples needed.

n — 樣本量

每組樣本數（unbalanced 設計需額外考慮 allocation ratio）。並非「越大越好」，要平衡資源、倫理與效應的臨床意義。

Per-arm sample size (consider allocation ratio for unbalanced designs). More is not always better — balance resources, ethics, and the clinical meaning of the effect.

💡
互相牽動：固定其中三個，第四個就唯一決定。最常見的「給定 α、β、效應 → 算 n」即此邏輯。 Mutual constraint: Fix any three, and the fourth is uniquely determined. The standard "given α, β, effect size → compute n" follows this logic.

實作範例

library(pwr)

# Two-sample t-test, Cohen's d = 0.5
pwr.t.test(d=0.5, sig.level=0.05, power=0.80,
           type="two.sample", alternative="two.sided")
# → n = 63.77 per group → round up to 64

# Two-proportion test: p1=0.40 vs p2=0.55
pwr.2p.test(h=ES.h(0.40, 0.55),
            sig.level=0.05, power=0.80)

# Survival: HR=0.6, two-arm balanced (epiR alternative)
library(powerSurvEpi)
ssizeCT.default(power=0.80, k=1, pE=0.6, pC=0.75,
                RR=0.6, alpha=0.05)
from statsmodels.stats.power import TTestIndPower, NormalIndPower
from statsmodels.stats.proportion import proportion_effectsize

# Continuous outcome — t-test
analysis = TTestIndPower()
n = analysis.solve_power(effect_size=0.5, alpha=0.05, power=0.80)
print(f"n per arm: {n:.1f}")  # 63.8

# Binary outcome — proportions
h = proportion_effectsize(0.40, 0.55)
n = NormalIndPower().solve_power(effect_size=h, alpha=0.05, power=0.80)
print(f"n per arm: {n:.1f}")

陷阱

三、常見錯誤

❌ post-hoc power

用觀察到的效應量回推檢定力沒有意義——它與 p 值一一對應，不能用來辯護「我的研究不顯著但有檢定力」。應該事前估計、並報告效應量信賴區間。

Computing power from the observed effect is meaningless — it is monotone with the p-value. It cannot defend a null result. Estimate prospectively and report effect size CIs.

❌ 忽略 dropout

實際追蹤的樣本通常會流失 10-30%。應在計算後乘以 1/(1-d) 補償。

Real cohorts lose 10-30%. Inflate by 1/(1-dropout rate) before recruiting.

❌ 忽略多重檢定

同時測 5 個 endpoint 卻仍用 α=0.05 → false positive 暴增。需以 Bonferroni 校正後 α 重新計算 n。

Testing 5 endpoints with α=0.05 inflates false positives. Recompute n using Bonferroni-adjusted α.

❌ 效應量靠想像

效應量需有依據：pilot study、文獻 meta-analysis、或臨床意義最小差異 (MCID)。空想的效應在 IRB 與審稿人面前難以辯護。

Effect size needs justification: pilot study, prior meta-analysis, or minimal clinically important difference (MCID). Made-up effects do not survive review.

📝 隨堂測驗

Q1. d 從 0.5 減半到 0.25，n 約變多少倍？

A. 2x
B. 4x
C. 8x
D. 不變No change
公式中 n ∝ 1/Δ²。效應量減半 → 樣本量約 4 倍。n ∝ 1/Δ². Halving the effect quadruples the needed sample.

Q2. GWAS 為什麼用 α=5×10⁻⁸？

A. 這是 IRB 規定It's an IRB requirement
B. 約等於 0.05 / 10⁶（百萬個獨立檢定的 Bonferroni）≈ 0.05 / 10⁶ — Bonferroni for ~1M independent tests
C. 是 Cohen 建議Cohen recommended it
D. 減少存檔大小Reduces file size
人類基因組約有 10⁶ 個獨立 LD block，Bonferroni 校正即得 5×10⁻⁸。The human genome contains ~10⁶ independent LD blocks; Bonferroni gives 5×10⁻⁸.

Q3. 為什麼不能用 post-hoc power 辯護不顯著？

A. 它違反 IRB 規定Violates IRB rules
B. 它與 p 值一一對應，不提供新資訊It is monotonic with the p-value — adds no information
C. 不能在 R 中計算Cannot be computed in R
D. 違反 GDPRViolates GDPR
post-hoc power 等同於 1 − p 值的單調函數，無法獨立證明檢定力是否足夠。應報告效應量信賴區間。Post-hoc power is monotone in p; it cannot independently demonstrate adequacy. Report effect-size CIs instead.
