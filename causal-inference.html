因果推論

從「相關」到「因果」需要額外假設與設計——DAG、傾向分數、工具變數、孟德爾隨機化。

Moving from "association" to "causation" needs extra assumptions and design — DAGs, propensity scores, IVs, Mendelian randomization.

概覽

為什麼需要因果框架？

傳統統計學的箴言：「相關不等於因果」。但生醫研究的目標往往就是因果——「服用 metformin 是否導致癌症風險下降？」。當隨機試驗不可行（倫理、成本、罕見結果）時，必須在觀察資料上做因果推論。

核心框架：Potential Outcomes (Neyman-Rubin) 與 Structural Causal Models (Pearl)。兩者本質等價，但 Rubin 框架偏重統計估計、Pearl 框架偏重圖論辨識。

"Correlation ≠ causation," but biomedical research is often about causation — "Does metformin cause reduced cancer risk?". When RCTs aren't feasible (ethics, cost, rare outcomes), causal inference on observational data is needed.

Two frameworks: Potential Outcomes (Neyman-Rubin) and Structural Causal Models (Pearl). Equivalent at the foundation; Rubin focuses on statistical estimation, Pearl on graph-theoretic identification.

DAG

一、DAG 因果圖

DAG 是一張箭頭圖：節點是變數，箭頭代表「直接因果」。透過 DAG 我們能用 back-door criterion 判斷需要調整哪些變數才能去除干擾偏倚。三種結構特別重要：

A DAG is an arrow diagram: nodes are variables, arrows are direct causal effects. The back-door criterion tells us which variables to adjust for to remove confounding bias. Three structures are crucial:

🔱 Confounder

A ← C → Y

C 同時影響 A 與 Y → 必須 調整 C 才能識別 A→Y 因果。

C affects both A and Y → must adjust for C to identify A→Y causal effect.

🌉 Mediator

A → M → Y

M 是 A 影響 Y 的中介。不要 調整 M（會切斷因果路徑、低估總效應）。

M lies on the causal path. Do NOT adjust for M (it blocks the path and biases the total effect).

🎯 Collider

A → K ← Y

K 是 A 與 Y 共同的下游結果。調整反而會引入假關聯。例：住院 = A(吸菸)+Y(肺癌) 的 collider。

K is a common effect. Adjusting creates spurious associations (e.g., hospitalization is a collider of A=smoking and Y=lung cancer).

💡
實用工具：用 dagitty.net 線上畫 DAG，會自動告訴你哪些變數要調整、哪些不要。 Useful tool: dagitty.net draws DAGs and automatically suggests which variables to adjust for.
傾向分數

二、傾向分數方法

當觀察性研究中「處置 (treatment)」並非隨機分配，傾向分數 (PS) 估計每個個體接受處置的條件機率：

PS(X) = P(T = 1 | X) — 通常用 logistic regression 或 ML 模型估

PS 後可用四種策略平衡兩組：

  • Matching：對每個處置者配對一名 PS 接近的對照（1:1 或 1:k）
  • Stratification：依 PS 分位數分層後比較
  • IPTW (Inverse Probability of Treatment Weighting)：給每人 1/PS 或 1/(1-PS) 權重
  • Covariate adjustment：把 PS 當作回歸共變數

檢驗 balance：配對後比較共變數的 Standardized Mean Difference (SMD)，< 0.1 為 well-balanced。

When treatment isn't randomly assigned, the propensity score (PS) estimates each subject's probability of being treated:

PS(X) = P(T = 1 | X) — fit with logistic regression or ML

Use PS to balance groups via four strategies:

  • Matching: match each treated with a control at similar PS (1:1 or 1:k)
  • Stratification: bin by PS quintiles, compare within each
  • IPTW: weight by 1/PS (treated) or 1/(1-PS) (control)
  • Covariate adjustment: use PS as a regression covariate

Check balance: compare covariates' Standardized Mean Difference (SMD); < 0.1 is well-balanced.

⚠️
PS 不能解決未測量的干擾因子。它只是把「觀察到」的共變數平衡到接近 RCT 的狀態。對未測量的混淆，需要工具變數或敏感度分析。 PS does NOT handle unmeasured confounders. It only balances observed covariates. For unmeasured confounding, use instrumental variables or sensitivity analyses (E-value).
工具變數

三、工具變數與孟德爾隨機化

工具變數 (Instrumental Variable, IV) 是滿足三項假設的變數 Z：

  1. 相關性：Z 影響暴露 A（Z 不能是無用的）
  2. 排除限制：Z 只能透過 A 影響 Y，無其他路徑
  3. 獨立性：Z 不受未觀察 confounder 影響

典型例子：經濟學的「征兵抽籤號碼」。在生醫上最重要的應用是孟德爾隨機化 (Mendelian Randomization, MR)——利用基因 variant 作為暴露（如 BMI）的 IV。

為什麼基因 SNP 是好的 IV？因為：

  • 受精時隨機分配（Mendel 第二定律）
  • 固定一生不被暴露反向影響
  • 與大部分混淆因子（社經、行為）獨立

常用 MR 方法：two-sample MR 結合 GWAS 結果（如 SNP→BMI 與 SNP→CHD 兩組獨立 cohort）。R 套件：TwoSampleMR、MendelianRandomization。

An Instrumental Variable (IV) Z satisfies three conditions:

  1. Relevance: Z affects exposure A
  2. Exclusion restriction: Z affects Y only through A
  3. Independence: Z is independent of unmeasured confounders

Canonical example: military draft lottery numbers. In biomedicine, the key application is Mendelian Randomization (MR) — using genetic variants as IVs for an exposure (e.g., BMI).

Why SNPs make good IVs:

  • Randomly assigned at conception (Mendel's 2nd law)
  • Fixed for life — exposure cannot reverse-cause genotype
  • Independent of most behavioral/socio confounders

Common MR: two-sample MR combines GWAS results (SNP→BMI and SNP→CHD from separate cohorts). R packages: TwoSampleMR, MendelianRandomization.

決策樹

如何選擇方法？

🌳 流程

Q1→可以隨機分配？是 → RCT（最佳）Can randomize? Yes → RCT (best)
Q2→所有干擾因子都觀察到？是 → 多變量回歸 / 傾向分數All confounders observed? Yes → multivariable regression / PS
Q3→有未觀察干擾但有 IV？是 → IV / 2SLS / MRUnmeasured confounding but have an IV? Yes → IV / 2SLS / MR
Q4→有自然實驗或政策變化？是 → Difference-in-Differences / Regression DiscontinuityNatural experiment / policy change? Yes → DiD / RDD
Q5→時間相依、有「治療-混淆-治療」結構？是 → G-methods (G-formula, MSM)Time-varying treatment with treatment-confounder feedback? Yes → G-methods (G-formula, MSM)

實作範例

library(MatchIt); library(cobalt)

# 1. Fit propensity score and match
m <- matchit(treat ~ age + sex + bmi + smoking,
             data=df, method="nearest", distance="logit", ratio=1)
summary(m)
love.plot(m, thresholds=0.1)   # SMD plot

# 2. Estimate treatment effect on matched data
matched <- match.data(m)
library(survey)
des <- svydesign(ids=~1, weights=~weights, data=matched)
svyglm(outcome ~ treat, design=des) |> summary()
library(TwoSampleMR)

# Exposure: BMI GWAS (Locke 2015); Outcome: CHD (CARDIoGRAMplusC4D)
exp <- extract_instruments("ieu-a-2")
out <- extract_outcome_data(snps=exp$SNP, outcomes="ieu-a-7")
dat <- harmonise_data(exp, out)

res <- mr(dat)   # IVW, MR-Egger, weighted median, etc.
print(res); mr_scatter_plot(res, dat)

# Sensitivity
mr_pleiotropy_test(dat)
mr_heterogeneity(dat)
from dowhy import CausalModel

model = CausalModel(
  data=df,
  treatment="treat",
  outcome="outcome",
  graph="""digraph {
    age -> treat; age -> outcome;
    sex -> treat; sex -> outcome;
    treat -> outcome;
  }"""
)
ident = model.identify_effect()
est   = model.estimate_effect(ident,
        method_name="backdoor.propensity_score_matching")
print(est.value)

# Refute estimate to check robustness
ref = model.refute_estimate(ident, est,
       method_name="placebo_treatment_refuter")
print(ref)

📝 隨堂測驗

Q1. A → M → Y 中要調整 M 嗎？

A. 應該調整Yes, adjust
B. 不應該調整 — M 是中介，會切斷因果路徑No — M is a mediator; adjusting blocks the path
C. 只在大樣本下調整Only with large n
D. 看 p 值決定Depends on p-value
中介變數調整會切斷 A→Y 的因果路徑，導致總效應估計偏低（甚至完全消失）。Adjusting for a mediator blocks the A→Y causal path, biasing the total effect toward zero.

Q2. 傾向分數的最大限制？

A. 計算太慢Slow computation
B. 無法處理未觀察的干擾因子Cannot handle unmeasured confounders
C. 只能用 logistic regressionOnly works with logistic regression
D. 不能配 IPTWIncompatible with IPTW
PS 只平衡「觀察到」的共變數，無法解決未觀察混淆。需配合敏感度分析（如 E-value）或 IV 方法。PS only balances observed covariates. Unmeasured confounding requires sensitivity analysis (E-value) or IV methods.

Q3. SNP 為什麼是好的 IV？

A. SNP 與表型相關性極高SNPs correlate strongly with phenotype
B. 受精時隨機分配且固定一生，與行為混淆獨立Randomly assigned at conception, fixed for life, independent of behavioral confounders
C. 每個 SNP 都是因果變異Every SNP is causal
D. SNP 直接影響疾病SNPs affect disease directly
SNP 的隨機分配與一生固定性，使它類似 RCT 中的「隨機處置」，避免反向因果與大部分混淆。Random allocation at conception and lifetime fixity mimic an RCT's random treatment, avoiding reverse causation and most confounding.

Q4. Collider bias 發生在？

A. 未調整干擾因子Failing to adjust for confounders
B. 調整了被兩變數共同影響的下游節點Adjusting for a common effect of two variables
C. 使用 Cox 模型Using a Cox model
D. 樣本量太小Small sample size
Collider 是兩變數共同的下游節點，調整它會在原本獨立的兩變數間引入假關聯。A collider is the common effect of two variables; conditioning on it induces a spurious association between them.
